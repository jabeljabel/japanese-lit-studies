# Japanese Literary Studies in North America
## A Digital Humanities Project

Interactive visualization suite for the essay
**"New Positivism, Same as the Old Positivism"**
Jonathan E. Abel, Penn State University

### About

This project maps 110 years of Japanese literary studies in North America
across seven interlocking datasets: 7,922 dissertations, 13,592 conference
presentations, 29,311 journal articles, citation networks, career trajectories,
and translation flows.

### Structure

- `index.html` — Browse all visualizations by branch, filter by topic, or take a guided tour
- `new_positivism_reader.html` — The essay with embedded interactive visualizations
- `networks/` — Advisor genealogies, authority networks, intellectual flow
- `content/` — Dissertation topics, journal trends
- `career/` — Job market and career trajectories
- `conferences_branch.html` — AJLS, AAS, MLA, ACLA, SCMS conference data

### Technical

All visualizations are self-contained HTML files using D3.js v7.
No server, no login, no installation required.
